# Resume Building Masterclass

Your resume is your first impression. If it fails the 6-second test, you lose the opportunity.

Today, we will build a resume that actually gets shortlisted.

---

# What Makes a Good Resume?

### 1. ATS (Applicant Tracking System) Compatibility
Most companies use software to scan resumes.
*   **Do:** Use standard fonts, simple bullet points, and clear headings.
*   **Don't:** Use complex tables, graphics, or multiple columns.

### 2. Action Verbs
Never start a bullet point with "Responsible for...".
Use strong verbs: **Managed, Developed, Spearheaded, Organized, Analyzed.**

### 3. The XYZ Formula
Accomplished [X] as measured by [Y], by doing [Z].
*Example:* "Increased club membership by 20% by organizing a campus-wide marketing campaign."

# Activity: Draft Your Resume

1.  **Format:** Pick a clean, single-column template (e.g., standard Google Docs template).
2.  **Contact Info:** Name, Professional Email, LinkedIn URL, Phone Number.
3.  **Education:** Latest degree first. Include CGPA if above 7.0.
4.  **Experience/Projects:** Use the XYZ formula for 2-3 bullet points each.
5.  **Skills:** Hard skills (tools, languages) and Soft skills.

**Task:** You have 30 minutes to draft the first version of your professional resume.
